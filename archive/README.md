# Retail Growth Advisor

This is a retail data mining project using local CSV files.

The system helps a retailer:
- find high-selling products
- find low-selling products
- decide what to stock up
- decide what to put on sale
- find product bundles using basket mining
- recommend products to customers
- predict product reorder behavior using classification models

## How to Run

Install packages:

pip install -r requirements.txt

Run app:

streamlit run app.py
